## WELLCOME \~!

SSAPI에 관심을 가져주셔서감사합니다. \
SSAPI는 아프리카 채팅/후원 등을 바탕으로 한 컨텐츠 및 프로그램 개발을 돕고자 제작되었습니다.



2024년 3월 초 오픈하여 현재 Private Beta 상태를 유지하고 있습니다.

* 아프리카TV의 경우 오랜 기간의 운영을 통해 어느정도 검증되었지만
* 치지직의 경우 아직 Beta 상태로 서비스 중 입니다.
