안녕하세요, SSAPI 운영팀입니다.

SSAPI는 2024년 2월 24일 출시 이후 현재로 **616일째**가 되었습니다. 그동안 많은 사용자분들로부터 접수된 의견으로 인하여 다양한 사항이 변경되게 되어 공지드립니다.

:::danger[⚠️ 중요 공지: API 키 재신청 필수]

**기존에 발급받으신 API 키는 2025년 11월 18일 23:59까지 Dashboard로 재신청해주셔야 합니다.**

### 📌 주요 안내사항

- **기존 발급된 API 키는 새로운 Dashboard와 호환되지 않아 신규 발급이 필요합니다.**
- **마감일**: 2025년 11월 18일 23시 59분
- **재신청 방법**: [SSAPI Dashboard](https://dashboard.ssapi.kr/)에서 신규 신청

### 🔑 기존 키 유지가 필요한 경우

이미 컨텐츠에 사용 중이셔서 키 변경이 곤란하신 경우, **문의하기를 통해 문의주시면 기존 키 그대로 마이그레이션을 도와드리겠습니다.**

**⚠️ 2025년 11월 18일 23시 59분 이후에는 기존에 발급된 모든 키가 삭제될 예정입니다.**

:::

## 🔓 1. API 키 신청 조건의 완화

기존 SSAPI에서 컨텐츠 제작 목적으로 발급을 신청하는 경우, 일정 이상 팔로워가 확인된 스트리머만 발급 신청이 가능했으며 운영진/개발자의 경우에도 해당 스트리머가 공식 인정하는 경우에만 발급을 허용해왔습니다.

### 🎯 주요 개선 사항

- 🌐 **간편한 신청**: 디스코드로 문의 주실 필요 없이 사이트 회원 가입 후 신청서를 작성해주시면 편리하게 신청이 가능합니다.
- ⚡ **즉시 테스트 가능**: 기존 방식의 경우 신청 후 심사가 완료되어야 테스트키가 발급되었지만, 현재는 **심사 중이더라도 테스트용 API 키가 자동으로 발급**되어 해당 키를 통해 바로 테스트가 가능합니다.

### 🔄 자동 관리 시스템

API 키 발급과 신청 관리, 자동 종료 등 기존 불필요하게 수동으로 관리되던 API 키는 이제 **Dashboard를 통해 자동으로 관리**됩니다.

- 🔑 **API 키 연장 및 신청서 변경**: 이제 Dashboard를 통해 간편하게 제출하시면 자동으로 관리되며, 별도 정기 감사가 진행되지 않을 예정입니다.
  - ⏰ **자동 연장 신청**: 특히 API 키 만료 14일 전의 경우, 대시보드를 통해 자동 연장 신청을 접수주시면 **별도 심사 없이 최대 한 달 범위로 자동 연장 신청이 가능**합니다.
- 👁️ **운영정책 모니터링**: 다만 운영정책 위반 유무에 대해서는 주기적으로 모니터링 중에 있으니 주의해주시기 바랍니다.

## 🎓 3. 치지직 API Beta 졸업

**치지직 API가 이제 베타를 졸업하여 정식 배포되었으며, 기존 방식에 비해 안정성이 다수 개선되었습니다.**

### 🔧 Beta 기간 중 확인된 문제

기존 치지직 API는 Beta로 운영되면서 일부 버그와 안정성 문제로 **데이터 수집이 누락되는 현상**이 있었습니다.

- 📉 **트래픽 집중 시간대**: 확인한 바에 의하면 트래픽이 몰리는 시간대에서는 **최대 2~8%의 데이터 누락**이 확인되었습니다.
- ✅ **개선 완료**: 정식 버전에서는 이러한 누락 현상이 대폭 개선되어 더욱 안정적인 데이터 수집이 가능해졌습니다.

그동안 베타 기간 동안 다양한 피드백을 주신 사용자분들께 감사드립니다. 정식 버전에서는 더욱 안정적이고 신뢰할 수 있는 서비스를 제공합니다.

## 🖥️ 4. 서버 증설

기존에도 **200개 이상의 키가 상시 발급**되어 서버 사용량이 항상 포화 상태였지만, 이제 변화된 대시보드와 변화된 정책으로 인하여 더 많은 키가 발급될 것으로 예상되어 **서버가 대폭 증설**되었습니다.

더 빠른 데이터 처리를 위해 **가상 서버에서 물리 서버로 이전**되었으며, 컴퓨팅 성능은 최대 2배, 메모리 성능은 최대 2배, 디스크 성능 또한 최대 25배 증가하여 더욱 안정적이고 빠른 서비스 제공이 가능해졌습니다. 특히 디스크 성능이 개선되어 쓰로틀에 의한 데이터 누락을 최소화했습니다.

- 📊 **추가 기간 통계**: 한 달 이상의 통계 정보가 필요하신 경우, **문의하기를 통해 심사 후 제공**해드립니다.

다만, 과도한 악용을 방지하기 위해 위와 같은 제한이 적용됩니다.
