안녕하세요, SSAPI 운영팀입니다.

SSAPI는 2024년 2월 24일 출시 이후 현재로 **616일째**가 되었습니다. 그동안 많은 사용자분들로부터 접수된 의견으로 인하여 다양한 사항이 변경되게 되어 공지드립니다.

<!--truncate-->

## 🔓 1. API 키 신청 조건의 완화

기존 SSAPI에서 컨텐츠 제작 목적으로 발급을 신청하는 경우, 일정 이상 팔로워가 확인된 스트리머만 발급 신청이 가능했으며 운영진/개발자의 경우에도 해당 스트리머가 공식 인정하는 경우에만 발급을 허용해왔습니다.

다만, 관련 내용에 대한 많은 불편 사항 접수로 인하여 **이제 컨텐츠 제작 목적이더라도 제품/프로그램을 신청하는 경우와 동일하게 대표 스트리머가 아닌 경우에도 신청이 가능하도록 완화**되었습니다.

### 📌 신뢰도에 따른 발급 제한

신청자의 신뢰도, 규모 등에 따라 발급 제한이 달라질 수 있습니다:

- ✅ **높은 신뢰도**: 대표 스트리머가 신청하고 대표 스트리머의 팔로워 수가 충분한 경우, 신뢰성이 높다고 판단하여 대용량 키 발급이 가능합니다.
- ⚠️ **일반 신뢰도**: 인증되지 않은 개발자분이 신청하시는 경우 등은 신뢰도가 낮다고 판단되어 리밋이 낮을 수 있습니다. (일반적으로 최대 20명 이내)

## 📝 2. API 키 신청 방식의 유연화

기존 API 키 신청은 디스코드로만 가능했으나, **이제는 새롭게 제작된 [SSAPI Dashboard](https://dashboard.ssapi.kr/)를 통해 가능**합니다.

### 🎯 주요 개선 사항

- 🌐 **간편한 신청**: 디스코드로 문의 주실 필요 없이 사이트 회원 가입 후 신청서를 작성해주시면 편리하게 신청이 가능합니다.
- ⚡ **즉시 테스트 가능**: 기존 방식의 경우 신청 후 심사가 완료되어야 테스트키가 발급되었지만, 현재는 **심사 중이더라도 테스트용 API 키가 자동으로 발급**되어 해당 키를 통해 바로 테스트가 가능합니다.

## 🎓 3. 치지직 API Beta 졸업

**치지직 API가 이제 베타를 졸업하여 정식 배포되었으며, 기존 방식에 비해 안정성이 다수 개선되었습니다.**

그동안 베타 기간 동안 다양한 피드백을 주신 사용자분들께 감사드립니다. 정식 버전에서는 더욱 안정적이고 신뢰할 수 있는 서비스를 제공합니다.

## 🖥️ 4. 서버 증설

기존에도 **200개 이상의 키가 상시 발급**되어 서버 사용량이 항상 포화 상태였지만, 이제 변화된 대시보드와 변화된 정책으로 인하여 더 많은 키가 발급될 것으로 예상되어 **기존 서버의 5배 사양으로 서버가 증설**되었습니다.

### 🚀 주요 개선 사항

- 📈 **대용량 키 발급 요건 완화**: 신뢰성이 충분히 확보되었다면 대용량 키에 대한 발급 요건이 대폭 완화될 예정입니다.
- 🛡️ **안정적인 서비스 제공**: 서버 증설로 인해 더욱 안정적인 서비스 제공이 가능해졌습니다.
- 🌟 **확장 가능한 인프라**: 앞으로 더 많은 사용자분들을 수용할 수 있는 환경이 마련되었습니다.

## 📊 5. API Dashboard 오픈

API 키 발급과 신청 관리, 자동 종료 등 기존 불필요하게 수동으로 관리되던 API 키는 이제 **Dashboard를 통해 자동으로 관리**됩니다.

### 🔄 자동 관리 시스템

- 🔑 **API 키 연장 및 신청서 변경**: 이제 Dashboard를 통해 간편하게 제출하시면 자동으로 관리되며, 별도 정기 감사가 진행되지 않을 예정입니다.
  - ⏰ **자동 연장 신청**: 특히 API 키 만료 14일 전의 경우, 대시보드를 통해 자동 연장 신청을 접수주시면 **별도 심사 없이 최대 한 달 범위로 자동 연장 신청이 가능**합니다.
- 👁️ **운영정책 모니터링**: 다만 운영정책 위반 유무에 대해서는 주기적으로 모니터링 중에 있으니 주의해주시기 바랍니다.

### ✨ 새로운 Dashboard 주요 기능

많은 컨텐츠 제작자분들께서 불편해하시던 기능들을 개선했습니다:

#### 📡 1) 실시간 데이터 모니터링

- API 키로 제공되는 데이터를 Dashboard에서 쉽게 확인할 수 있는 **실시간 데이터 모니터링 페이지**가 추가되었습니다.
- 이 페이지를 통해 API에서 제공되는 데이터를 실시간으로 확인 가능합니다.

#### 🧪 2) 후원 테스트 및 API 복구 기능

- 기존에는 후원이 잘 들어오는지 확인하려면 관리자 전용 API 복구 기능을 관리자 심사를 통해 오픈해드리고 있었습니다.
- 이제 **실시간 데이터 모니터링 페이지를 통해 사용자분들께서 간편하게 후원 테스트 기능과 API 복구를 수행 가능**합니다.
- ⚠️ 이로 인해 기존 제공되던 **관리자용 후원 복구 명령어 기능은 삭제**되었습니다.

#### 📈 3) 후원 통계 모니터링

- 많은 컨텐츠 개발자분들이 궁금해하시는 "어떤 API가 후원이 많이 들어오는지" 현황을 확인할 수 있는 **후원 실시간 모니터링 페이지**를 추가했습니다.
- 기존에는 대형 프로젝트에 한하여 별도 커스터마이징하여 제공했으나, 이제 **Dashboard에서 누구나 최근 한 달 현황에 대해 간편하게 확인 가능**합니다.

## 💾 6. 데이터 보존 기간 변화

기존에는 SSAPI는 **후원 데이터를 최대 3일간 보관**했으나, 이제 **최대 1년간 보관**될 수 있습니다.

이는 컨텐츠 제작자분들께 **컨텐츠 제작 결과에 대한 성과 리포트를 요청주시는 경우 제공**해드릴 수 있도록 변화된 것입니다.

### 📋 데이터 보관 정책

- 🗄️ **SSAPI 서버**: 최대 1년간 보관
- 👤 **사용자 접근**: 개정된 정책에 따라 **최대 한 달간 보관 및 이용** 가능
- 📊 **추가 기간 통계**: 한 달 이상의 통계 정보가 필요하신 경우, **문의하기를 통해 심사 후 제공**해드립니다.

다만, 과도한 악용을 방지하기 위해 위와 같은 제한이 적용됩니다.

## ⭐ 7. 새로운 등급 '파트너' 추가

기존에는 **제휴사 등급**만 존재했지만, 제휴사 등급은 조건이 까다로웠습니다.

이제 **API 키를 자주 사용하시는 제작자분들에 한하여 신뢰성이 확보된 경우, 파트너 등급으로 편의성을 제공**해드리고 있습니다.

### 🎁 파트너 등급의 혜택

- 🚀 **API 키 무심사 발급**: SSAPI를 장기간 사용하며 신뢰성이 확보된 경우, API 키 발급 시 심사 없이 사용 신청서만 제출하면 즉시 발급됩니다.
- 📜 **운영정책 전체 준수**: 제휴사와 달리 운영정책을 모두 준수하되, 간소화된 절차로 빠르게 서비스를 이용하실 수 있습니다.

자세한 등급 안내는 [함께하는 프로젝트 페이지](https://ssapi.kr/docs/projects#파트너-등급-안내)에서 확인하실 수 있습니다.

## 📖 8. 문서 및 프로젝트 페이지 개선

API 문서 페이지를 조금 더 보기 편하도록 재구성하고 가독성을 개선했습니다. 또한 연말을 맞이하여 [함께하는 프로젝트 페이지](/docs/projects)를 최신화하고 새롭게 추가된 파트너에 대한 정보를 추가했습니다.

### 📋 주요 개선 사항

- 📐 **사이드바 최적화**: 좌측 사이드바 너비를 조정하여 콘텐츠 영역을 확보하고 메뉴 가독성을 향상시켰습니다.
- 🆕 **신규 파트너 추가**: 새롭게 함께하게 된 파트너사들의 정보를 추가했습니다.
- 🔄 **프로젝트 목록 최신화**: 현재 진행 중인 프로젝트들의 최신 정보를 반영했습니다.
- 📊 **파트너 등급 안내 UI**: 제휴사, 파트너, 일반 사용자 등급에 대한 안내를 카드 형태로 시각화했습니다.

더욱 풍성해진 SSAPI 생태계를 [프로젝트 페이지](/docs/projects)에서 확인해보세요!

## 🚀 9. 앞으로의 계획

SSAPI는 계속해서 발전하고 있습니다. 다음은 현재 진행 중이거나 곧 출시될 주요 업데이트 내용입니다:

### 📺 치지직 미션 및 숲 도전미션/대결 지원

치지직의 **미션**과 숲의 **도전미션/대결**에 대한 지원을 준비 중이며, 이 작업은 **2주 이내로 진행 예정**입니다. 더욱 다양한 스트리밍 기능을 API를 통해 활용하실 수 있게 됩니다.

### 🎮 마인크래프트 플러그인 1.1 개발

많은 개선 요청이 있었던 마인크래프트 플러그인의 새 버전 개발에 착수했습니다:

- 🔧 **기능 추가 제약 개선**: 더욱 유연한 플러그인 확장이 가능해집니다.
- 🎁 **아이템 지급 커맨드 다중 설정**: 기존에는 하나만 설정 가능했던 아이템 지급 커맨드를 여러 개 설정할 수 있도록 개선됩니다.
- 📢 **추후 공지 예정**: 플러그인 1.1 버전에 대한 자세한 내용은 추후 별도로 공지드리겠습니다.

### 🎥 유튜브 플랫폼 지원

유튜브 플랫폼에 대한 지원을 위해 개발을 진행 중입니다. 해당 내용은 **추후 공지 예정**입니다.

### 📊 구독 및 팔로워 수 증가 API 지원 검토

구독과 팔로워 수 증가 등에 대한 API 지원을 검토 중입니다. 스트리머분들의 성장 지표를 더욱 쉽게 추적하고 활용하실 수 있도록 준비하고 있습니다.
