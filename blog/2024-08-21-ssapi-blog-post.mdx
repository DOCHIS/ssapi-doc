# 숲(아프리카)/치지직 채팅 및 후원 API 서비스

안녕하세요, 개발자 헛삯입니다.

SSAPI는 숲(아프리카)과 치지직 플랫폼에서 후원 및 채팅 데이터를 제공하는 API 서비스입니다. **최적화된 설계와 체계적인 문서화**를 통해 누구나 쉽게 사용할 수 있으며, 대규모 서버에서도 안정적으로 운영될 수 있도록 개발되었습니다.

---

## <!--truncate-->

## SSAPI의 성과

2024년 2월 24일에 오픈한 이후 단 6개월 만에 **150억 건 이상의 API 데이터를 제공**하였으며, 약 1,000명의 스트리머 분들께서 이용해주셨습니다. 또한, 약 30명의 개발자와 10개 이상의 마인크래프트 서버, 20여 개의 콘텐츠 개발 프로젝트와 협력 중입니다.

---

## SSAPI의 비전과 목표

콘텐츠 제작에는 많은 비용과 노력이 필요합니다. 후원을 기반으로 한 콘텐츠 제작은 제작비를 충당하는 중요한 수단입니다.

SSAPI는 더 많은 스트리머가 부담 없이 콘텐츠를 제작하고 방송 생태계를 풍부하게 만들 수 있도록 지원합니다. 이를 위해 지속적인 최적화와 개발자와의 소통에 중점을 두고 있으며, 대규모 참여자 기반 콘텐츠에서도 원활하게 사용될 수 있도록 최고 수준의 성능을 목표로 합니다.

---

## SSAPI 활용 사례

- **A사례**: 기존 시스템의 API 누락 문제를 해결하고 안정적인 서버 운영에 기여.
- **B사례**: 서버 부하로 인해 별도의 API 전용 서버를 사용하던 상황에서 비용 절감 성공.
- **C사례**: API 네트워크 사용량 감소로 전용 회선에서 일반 회선으로 전환 성공.

---

## SSAPI와 함께하는 프로젝트

### 제휴 / 파트너

- **소프트콘 뷰어십**: 데이터 일부를 제휴 형태로 제공 (2024년 05월 23일~)
- **뻐스시간**: 버튜버 예능 그룹과 콘텐츠 제작 협업 (2024년 03월 10일~)

- **클로배**: 2024년 05월 15일~, 약 60명의 스트리머 참여.
- **힐링월드**: 2024년 06월 01일~, 약 80명의 스트리머 참여.

## 사용 범위와 정책

SSAPI는 서드파티 API 서비스로, 제공받은 데이터의 보관 기간과 사용 범위에 엄격한 제한이 있습니다.

- **데이터 보관 기간**: 3일 초과 불가.
- **사용 제한**: 스트리머 동의 없는 데이터 사용은 불허.
- **위반 시**: API 키는 사전 통보 없이 삭제.
