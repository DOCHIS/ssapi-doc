## ✨ 1주년 기념 API 문서 개선

**API 버그 수정 및 문서 리뉴얼**: https://ssapi.kr

SSAPI 1주년을 맞이하여 특별한 업데이트 소식을 전해드립니다! 🎉

API 관련 이슈를 해결하고, 1주년을 기념하여 API 문서 시스템을 전면 개선하였습니다:

- 폴링 API 호출 시 일부 API 키에서 발생하던 "입력값 오류" 문제가 해결되었습니다
- API 문서 사이트가 전면 리뉴얼되어 https://ssapi.kr 주소로 제공됩니다
- REST API를 웹사이트에서 직접 테스트해볼 수 있는 UI가 추가되었습니다
- Socket API와 REST API의 다양한 프로그래밍 언어 사용 예제가 추가되었습니다
- 문서를 더욱 깔끔하게 볼 수 있도록 다양한 편의 기능이 개선되었습니다

## 📊 서비스 현황

SSAPI는 지난 1년간 놀라운 성장을 이루었습니다:

- **2,500+** 함께한 스트리머 수
- **500억+** 누적 API 호출량
- **100+** 활성 개발자 수
- **3+** 제휴 파트너사
- **20+** 함께 한 게임서버
- **99.999%** 서비스 가동률

## 🤝 새로운 제휴 소식

- **(주)일리오**: 2025년 1월 14일부터 셀럽후원플랫폼 팬심과 함께 더 나은 방송 생태계 구축을 위한 데이터 공유 제휴를 체결하였습니다.

## 🎮 신규 마인크래프트 프로젝트

현재 SSAPI와 함께하는 주요 마인크래프트 프로젝트들을 소개합니다:

- **마병대2**: 2025년 01월 24일 ~ (약 100명의 스트리머 참여)
- **다이아?서버**: 2025년 01월 4일 ~ (약 150명의 스트리머 참여)
- **청새치숲4**: 2024년 12월 1일 ~ (약 270명의 스트리머 참여)
- **요뚜기월드 시즌2**: 2024년 10월 16일 ~ (약 140명의 스트리머 참여)

## 🚀 서비스 확장 소식

최근 API를 활용한 서비스와 제품 개발에 대한 제휴 문의가 증가함에 따라, 이용정책을 개정하여 대형 게임 서버뿐만 아니라 다양한 방송 생태계 발전을 위한 프로그램 개발도 적극 지원하고 있습니다.

## 🎂 SSAPI 1주년 안내

2025년 2월 1일, SSAPI가 드디어 1주년을 맞이합니다!

2024년 2월 1일 SSAPI 출시 이후, 수많은 스트리머분들과 개발자분들의 사랑을 받아왔습니다. 이를 기념하여 API 문서의 대규모 개선을 진행하였으며, 특히 다양한 언어의 예제 추가로 더욱 쉽게 사용하실 수 있습니다.

그동안 SSAPI를 사랑해주신 모든 분들께 진심으로 감사드립니다. 앞으로도 더욱 안정적인 서비스와 새로운 기능으로 여러분의 성원에 보답하도록 하겠습니다! ❤️

더 자세한 내용은 [공식 사이트](https://ssapi.kr)에서 확인하실 수 있습니다.

감사합니다! 🙏
